# Artificial-Intelligence-Lab

This repository contains lab assignments, projects, and experiments completed as part of the **Artificial Intelligence (AI) Lab** course.

## Course Objectives

- Understand fundamental AI concepts
- Implement AI algorithms and techniques
- Solve real-world problems using AI
- Gain hands-on experience with AI tools and libraries

## Repository Structure

```
├── Lab-01/
├── Lab-02/
├── Lab-03/
├── Project/
└── README.md
```

## Technologies

- Python 3.x
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/samshawon10/ai-lab.git
   ```

2. Navigate to the project:
   ```bash
   cd ai-lab
   ```

3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```
